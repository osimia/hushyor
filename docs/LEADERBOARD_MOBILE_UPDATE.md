## 🎨 Дизайн решения:

### Проблема:
На мобильных устройствах текст в списке рейтинга сливался из-за таблицы с 4 колонками.

### Решение:
- Создан отдельный мобильный layout с вертикальным расположением
- Использованы классы `hidden md:grid` для desktop и `md:hidden` для mobile
- Применен `flex` layout для горизонтального выравнивания элементов
- Добавлен `truncate` для обрезки длинных имен

## 📱 Адаптивные breakpoints:

- **Mobile**: < 768px - вертикальный компактный вид
- **Desktop**: ≥ 768px - таблица с 4 колонками
