## Команды для импорта

### 1. Проверка данных (dry-run):
```bash
python manage.py import_math_from_json math_tests_import.json --dry-run
```

### 2. Импорт в БД:
```bash
python manage.py import_math_from_json math_tests_import.json
```

### 3. Импорт с очисткой старых данных:
```bash
python manage.py import_math_from_json math_tests_import.json --clear
```

## Важно!

- Предмет "Математика" должен существовать в БД с ID=2
- Команда автоматически создаст темы и тесты
- Если тест с таким `original_test_id` уже существует, он будет пропущен
- Используйте флаг `--clear` только если хотите полностью заменить все тесты по математике

## Проверка после импорта

1. Зайдите в Django admin: `/admin/`
2. Откройте раздел "Tasks" (Тесты)
3. Отфильтруйте по предмету "Математика"
4. Проверьте количество тестов (должно быть 681)
5. Проверьте темы в разделе "Topics"
